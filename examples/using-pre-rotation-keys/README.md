# Using Pre-Rotation Keys (Example)

The DID Toolbox also supports a [_post-quantum safe_](https://didwebvh.info/latest/implementers-guide/prerotation-keys/#post-quantum-attacks)
technique called [Key Rotation with Pre-Rotation](https://didwebvh.info/latest/implementers-guide/prerotation-keys).

The code examples in this directory should illustrate using the DID Toolbox Java API
for the purpose of creating DID logs featuring pre-rotation keys.

## Project integration

To be able to reuse snippets from the code example, please accommodate your `pom.xml` accordingly - 
[any version from 1.8.0 onwards](https://repo1.maven.org/maven2/ch/admin/swiyu/didtoolbox)
should work like a charm:
```xml
<!-- file: pom.xml -->
<dependencies>

    <dependency>
        <groupId>ch.admin.swiyu</groupId>
        <artifactId>didtoolbox</artifactId>
        <!--version>[ANY_AVAILABLE_VERSION_FROM_1_8_0_ONWARDS]</version-->
    </dependency>

    <!-- Other dependencies go here -->

</dependencies>
```

## Running the example

To produce a valid (`did:webvh:1.0`) DID log featuring pre-rotation keys, just run:
```shell
mvn -q compile exec:java
```
or (if `jq` command is available):
```shell
mvn -q compile exec:java | jq
```
